# Project Mesto Full

## Description
Repository for study project at Yandex.Practicum. Includes both frontend and backend parts. Single-page application for posting, liking and deleting image cards with registration and authorization.

## Stack frontend
- HTML
- CSS
- JS
- React Js
- REST API

## Stack backend
- Node Js
- Express
- Mongo Atlas DB
